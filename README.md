# IQ Snake Game

Classic snake game with on-chain leaderboard using IQ Labs SDK.

## Setup

```bash
cp .env.example .env
npm install
npm start
```

Open http://localhost:3000

## Requirements

- Solana keypair at `~/.config/solana/id.json` (or set `SOLANA_KEYPAIR_PATH`)
- Devnet SOL for transaction fees
